rtek-footer Component

A generic footer web component. Inherits all styles from parent.

Installation

<script src="https://cdn.jsdelivr.net/npm/rtek-footer"></script>

Usage

Wrap the component in a styled container:

Dark theme (centered)
<div style="background: #0f172a; color: white;">
  <rtek-footer
    company="Expert Partner AS"
    founded="2020"
    instagram="https://instagram.com/epklima"
    facebook="https://facebook.com/epklima"
    developer="r-TEK.io"
    developer-url="https://r-tek.io"
  ></rtek-footer>
</div>
Gold theme (centered)
Left-aligned
Minimal (no socials, no developer)

Attributes

company       - Company name
founded       - Year founded (shows range if different from current year)
facebook      - Facebook URL
instagram     - Instagram URL
x             - X (Twitter) URL
snapchat      - Snapchat URL
linkedin      - LinkedIn URL
developer     - Developer/agency name
developer-url - Developer website URL
align         - Content alignment: start, center (default), end
