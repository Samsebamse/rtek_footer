rtek-footer

Generic footer web component with optional brand, contact, hours, social links, and copyright sections. Backward compatible — omit new attributes for the original minimal footer.

Installation

<script src="https://cdn.jsdelivr.net/gh/Samsebamse/rtek_footer@v2.0.0/src/rtek-footer.js"></script>

Full Footer (Pelinis style)

Brown + gold theme — brand, contact, hours, social, copyright

Full Footer (Bymico style)

Dark navy + blue accent theme

Minimal (backward compatible)

Social + copyright only — no grid sections

Attributes

Brand
  logo          - Logo image URL
  logo-height   - Logo height (default: 6rem)
  tagline       - Description text below logo

Contact
  contact-title - Section heading (e.g. "Contact")
  phone         - Phone number (auto tel: link)
  email         - Email address (auto mailto: link)
  address       - Pipe-separated lines (e.g. "Street|City|Country")
  address-url   - Google Maps URL for address link

Hours
  hours-title   - Section heading (e.g. "Hours")
  hours         - JSON: [{"day":"Mon-Fri:","time":"09-17"},{"day":"Sun:","time":"Closed","closed":true}]

Social
  facebook, instagram, tiktok, x, snapchat, linkedin, whatsapp

Theming
  bg-color      - Footer background color
  color         - Text color
  accent-color  - Headings, icons, hover, "closed" label
  border-color  - Top border + bottom bar border

Layout
  company       - Company name (copyright)
  founded       - Year founded (year range)
  developer     - Developer name
  developer-url - Developer website
  align         - start | center (default) | end
  font-size     - Base font size
  social-gap    - Gap between social icons
  gap           - Gap between social row and copyright
